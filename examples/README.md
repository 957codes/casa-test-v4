# Examples

This folder shows what Casa produces without requiring you to install anything.

## inboxpilot/ (a full, real company brain)

A complete, committed company brain for InboxPilot, a fictional idea-stage company (a Chrome extension that summarizes long email threads and drafts replies). Every file was generated by the real Casa engine, not hand-mocked. This is the standard of output Casa holds itself to.

How to browse it:

- `company-brain/NOW.md` is what the founder sees at the start of every session: the current level, the north star, the do-or-die constraint, the single next action, and what can run in parallel.
- `company-brain/build-map.json` is the personalized plan: 107 of the 169 playbooks selected for this business type and traits, sequenced by the dependency graph.
- `company-brain/outputs/opportunity-scan/opportunity-brief.md` is a real graded artifact: the Level 0 Opportunity Scan, written to the playbook's deliverable spec and rubric.
- `company-brain/scores.jsonl` is the grade for that artifact, in the shape casa-review writes: a score, a pass verdict, and the named gap that remains. Casa records what is still weak instead of calling everything done.
- `company-brain/CLAUDE.md` is the self-updating company contract: the operating rules plus AUTO blocks that the engine rewrites as work completes (you can see it already reflects one done play and the next action).
- `company-brain/answers.json` is the stage-interview output that seeded all of the above.
- `company-brain/profile.json`, `pulse.json`, `roster.json`, and `dials.json` are the derived business profile, founder priorities, department roster, and autonomy dials.

## Profile JSONs (try the router directly)

`profile-b2b-devtool.json` and `profile-solana-analytics.json` are router inputs. From the repo root:

```
node scripts/router.mjs plan examples/profile-b2b-devtool.json
node scripts/router.mjs next examples/profile-solana-analytics.json
```

`plan` prints the full personalized build map with selection reasons. `next` prints the ranked next actions for that business at its current level.
