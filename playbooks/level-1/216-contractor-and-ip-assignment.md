# Contractor and IP Assignment Agreements

## Procedure
1. List every contributor. Build a single sheet of everyone who has touched code, design, copy, logos, or smart contracts since inception, including founders, freelancers, agencies, and pre-incorporation help. Note what each produced and whether anything is signed today.
2. Get founder IP assignment signed first. Have each founder sign a confidential information and invention assignment agreement (CIIAA) that assigns all prior and ongoing work product to the company. This is the document investors check before the term sheet.
3. Adopt a standard contractor agreement template. Use a work-for-hire plus present-assignment clause ("hereby assigns"), confidentiality, and a moral-rights waiver. Send it to every active and past contractor and agency for signature. For pre-incorporation work, add a confirmatory assignment so the new entity inherits clean title.
4. Cover employees and future hires. Make the CIIAA a mandatory part of every offer so no one starts before signing. Keep one canonical template per role type.
5. Have counsel review and execute. Run the founder and contractor templates past a startup lawyer once, then use e-signature for all parties. Store countersigned PDFs in one folder.
6. Close gaps in the register. Record each signed agreement, counterparty, and date. Any open-source license obligations or third-party assets get logged so ownership claims hold up in diligence.

## Output
ip_assignment_agreements signed by all contributors, a reusable contractor_agreement_template, and an ip_ownership_register ready for fundraising diligence.
